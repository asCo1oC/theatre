# Theatre Telegram Bot

**Внимание!** Это учебный (тренировочный) Telegram-бот для автоматизации бронирования билетов в театр. Использовать его для реальных покупок нельзя — он предназначен только для демонстрации и обучения.

---

## Как запустить

1. Клонируйте репозиторий:
   ```bash
   git clone <адрес_репозитория>
   cd theatre
   ```
2. Создайте файл `.env` с вашим токеном Telegram-бота и настройками:
   ```env
   TG_BOT_TOKEN=ваш_токен
   # Остальные переменные по необходимости
   ```
3. Соберите и запустите контейнер:
   ```bash
   docker compose build
   docker compose up -d
   docker compose logs -f ogatt-booker
   ```
4. Для остановки:
   ```bash
   docker compose down
   ```

---

**Этот бот не предназначен для реального бронирования билетов. Используйте только в тестовых целях!**
